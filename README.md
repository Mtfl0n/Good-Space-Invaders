## Controls
- **Left/Right Arrows** — ship movement
- **Space** — shoot
- **P** — pause
- **Esc** — exit

## Features
- Multiple difficulty levels
- Adjustable volume level (Broken/Unfinished)
- Select game background in settings
- Save best scores
- Pause during game (P key)
